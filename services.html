Services

Brand Identity

Creating a brand identity is crucial these days to build up brand awareness and a loyal customer base.

Using various tools I can provide the creation of a brand identity kit including your logo, colour palette and fonts. This kit and all the assets are then required in the process of designing and developing the website.

Web Design

Before creating a website, the first stage after finalising the logo, fonts and colour palette is to create a visual of your website.

Using tools like Figma, I can provide you with examples of your website across multiple devices and including the flow of user experience.

Web Development

Once you have decided on the design of your website, it is now time to create the real thing.

With experience in Wordpress, Webflow and Squarespace I can take a design and create a website that is secure, fully responsive and SEO optimised.

If you would like to discuss a project then please drop me an email!
